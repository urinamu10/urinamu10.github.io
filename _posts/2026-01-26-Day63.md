####  121. Patients With a Condition

출처 -https://leetcode.com/problems/patients-with-a-condition/

- 테이블 

![alt text](/assets/img/image107.png)


- **문제** 

> 제1형 당뇨병 환자의 patient_id, patient_name, 상태를 찾기 위한 해결책을 작성하세요. 
> 제1형 당뇨병은 항상 접두사로 시작합니다.DIAB1
> 결과 표를 어떤 순서로든 반환하세요.결과 형식은 다음 예시와 같습니다.
{: .prompt-tip }

![alt text](/assets/img/image108.png)


- 풀이 
> **1. condition 컬럼에서 `DIAB1`이 포함**

당뇨병 환자의 환자id, 환자이름, 상태를 patients 테이블이 가지고있는데
문제는 해당 테이블의 상태(conditions) 컬럼에서 `DIAB1`을 포함한 글자만 남겨두어야하므로 `like` 문을 사용하여 해결 할 수 있었습니다.


```sql
select *
from patients 
where conditions like '%DIAB1%';
```

##### 122. Delete Duplicate Emails

출처 - https://leetcode.com/problems/delete-duplicate-emails/description/


- 테이블 

![alt text](/assets/img/image109.png)

- **문제**
> 중복된 이메일을 모두 삭제하고, 가장 작은 id.을 가진 고유 이메일은 기준 하나만 남기는 솔루션을 작성하세요.
> SQL 사용자분들은 DELETE를 이용해 중복을 제거해야합니다
> 결과 형식은 다음 예시와 같습니다.
{: .prompt-tip }

- 풀이 
> **1. 중복을 제거하기** 

같은 이메일을 가진 행중, id가 더 큰 행을 삭제하여 문제를 해결하였습니다.

```sql
Delete p1
from person p1, person p2
where p1.eamil = p2.email and p1.id > p2.id ;
```
