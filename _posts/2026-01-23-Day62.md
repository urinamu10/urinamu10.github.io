#### 120. Fix Names in a Table

출처 - https://leetcode.com/problems/fix-names-in-a-table/description/

- 문제 

> 이름을 고정해 첫 문자만 대문자이고 나머지는 소문자로 설정하는 솔루션을 작성하세요.
>로 정렬된 결과 테이블을 반환합니다.user_id
>결과 형식은 다음 예시와 같습니다.


<details>
<summary> 복잡한 SQL 쿼리 보기</summary>

```sql
SELECT user_id, count(*) FROM orders GROUP BY 1;
```
</details>


>[!TIP]
>**1. 풀이 과정**
>데이터 전처리를 위해 Pandas의 `dropna()`를 활용했습니다.

>[!INFO]
>**2. 참고 사항**
>최종 프로젝트 도메인인 E-커머스 분석 시 유의할 점입니다.


>[!WARNING]
>**3. 위험 사항**
>위험한 상황일 떈 이런 방식으로 출력됨
