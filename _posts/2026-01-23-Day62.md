#### 120. Fix Names in a Table

출처 - https://leetcode.com/problems/fix-names-in-a-table/description

- **테이블**
![alt text](/assets/img/image103.png)


- **문제 설명**

> 이름을 고정해 첫 문자만 대문자이고 나머지는 소문자로 설정하는 솔루션을 작성하세요.
> ```user_id``` 기준으로 오름차순 정렬하여 테이블을 반환합니다
> 결과 형식은 다음 예시와 같습니다.
{: .prompt-info }

![alt text](/assets/img/image106.png)


- 풀이 

> 1. concat 함수로 첫번째 + 나머지 결합
`upper`와 `substr`으로 `name` 컬럼의 첫글자만 대문자로 변환하고, 2번째부턴 `lower` 함수로 소문자로 변환한 후 결합하는 방식으로 문제를 해결하였습니다.

```sql 
select user_id,
       concat(upper(substr(name,1,1)), lower(substr(name,2))) as name
from users 
order by user_id ;
```
