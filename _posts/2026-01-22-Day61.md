> 2. 조건문으로 a값을 list에 중첩시키기
>
비어있는 리스트를 하나 만들어주고
1번의 반복문 속 조건문을 설정합니다.
- 만약 b가 `True`라면 `a`를 리스트에 추가
- 만약 b가 `false`라면 `-a`를 리스트에 추가

```python
if b == True:
    list.append(a)
else:
    list.append(-a)
```

> 3. 리스트의 합을 return 하기 

최종적으로 누적시킨 리스트의 합계를 반환하는 함수를 완성할 수 있었습니다.


```python
def solution(absolutes, signs):
    
    list = []
    for a,b in zip(absolutes, signs):
        if b == True:
            list.append(a)
        else:
            list.append(-a)
    return sum(list)

print(solution([4,7,12], [True, False, True]))
```
